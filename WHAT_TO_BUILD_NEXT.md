### **Option A: Course Management** 🎓
**Build the core LMS functionality**

#### For Teachers:
1. **Create Course**
   - Form to create new courses (title, description, syllabus)
   - Set course status (draft, published, archived)
   - View list of their courses

2. **Manage Course**
   - Edit course details
   - Upload course materials (PDFs, videos, links)
   - View enrolled students
   - Delete courses

#### For Students:
1. **Browse Courses**
   - View all published courses
   - Search and filter courses
   - See course details before enrolling

2. **Enroll in Courses**
   - One-click enrollment
   - View enrolled courses
   - Access course materials

**Complexity:** Medium  
**Impact:** High - Core LMS feature
